# Scoop Bucket

Bucket for [Scoop](https://scoop.sh) containing manifests for installing tools
and applications that I have created.

## Usage

Add this bucket to Scoop and then install the package(s) you would like.

```
scoop bucket add markembling https://github.com/markembling/scoop-bucket
scoop install markembling/<package>
```
